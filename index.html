Function Composition

It's easy to compose functions with rubico, just pipe them together. If you need to act on collections, e.g. an array, reach for map and filter. In this example, we declare squaredOdds from square and isOdd.

By running the example as is, we expect the input array [1, 2, 3, 4, 5] to transform into the output array [1, 9, 25].
